# WTA

A GRASP-based approximate one-step rollout constructive heuristic for the time-windowed multi-burst WTA, where the rollout evaluation is approximated in 
O(n) per candidate via an exact kill-chain capacity-destruction surrogate, followed by a schedule consolidation phase and an Enriched Iterated Greedy metaheuristic with RLS local search.

Scheduling: Parallel Machine Scheduling with Time Windows and Resources Constraint (PMSP-TW), Objective: Stochastic Quiz Problem with Time Windows
